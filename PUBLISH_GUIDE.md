# 应用商店发布指南

## 软件基本信息

- **应用名称**: 文章多平台发布助手
- **开发者**: 合果科技
- **官网**: www.hergo.com.cn
- **版本**: 1.0.0

## 各平台发布步骤

### 1. Windows 应用商店 (Microsoft Store)

#### 准备工作
1. 注册 [Microsoft 合作伙伴中心](https://partner.microsoft.com/) 账号
2. 完成开发者注册（个人 $19，公司 $99）

#### 打包步骤
```bash
# 构建应用
npm run build

# 打包为 MSIX
npm run dist:win
```

#### 提交要求
- 应用图标: 多种尺寸 (71x71, 150x150, 310x150, 310x310, 44x44)
- 截图: 1366x768 或 1920x1080
- 应用描述
- 隐私政策链接

---

### 2. macOS App Store

#### 准备工作
1. 注册 Apple Developer Program ($99/年)
2. 获取开发者证书

#### 打包步骤
```bash
# 构建应用
npm run build

# 打包为 DMG 和 PKG
npm run dist:mac
```

#### 提交要求
- 应用图标: 1024x1024
- 截图: 1280x800, 1440x900, 2560x1600, 2880x1800
- 应用描述
- 隐私政策链接

---

## 所需材料清单

### 必备材料
1. ✅ 软件著作权证书（国内商店需要）
2. ✅ ICP 备案（如有官网）
3. ✅ 企业营业执照
4. ✅ 应用图标（多种尺寸）
5. ✅ 应用截图（3-5张）
6. ✅ 应用描述文案
7. ✅ 隐私政策页面
8. ✅ 用户协议

### 应用描述文案模板

**简短描述（80字内）:**
一键将文章发布到微信公众号、知乎、今日头条等多个平台，提升内容分发效率。

**详细描述:**
文章多平台发布助手是一款专为内容创作者设计的效率工具。

主要功能：
- 支持15+主流平台一键发布
- 简洁高效的文章编辑器
- 自定义平台扩展
- 发布历史追踪
- 本地数据安全存储

支持平台：微信公众号、知乎、今日头条、百家号、简书、博客园、CSDN、掘金等。

开发者：合果科技
官网：www.hergo.com.cn

## 注意事项

1. **软件著作权** - 国内应用商店通常需要软著证书，建议提前申请（约1-2个月）
2. **隐私政策** - 必须提供隐私政策页面，说明数据收集和使用情况
3. **用户协议** - 建议提供用户协议
4. **安全检测** - 部分商店会扫描代码，确保没有恶意代码
5. **更新维护** - 定期更新版本，修复bug
