# 文章多平台发布助手

一款简洁高效的多平台文章一键转发工具，支持微信公众号、知乎、今日头条、百家号等主流平台。

## 功能特点

- 🚀 **多平台一键发布** - 同时发布到多个平台，省时省力
- ✏️ **简洁编辑器** - 专注写作，所见即所得
- 🔧 **自定义平台** - 支持扩展更多平台
- 📊 **发布历史** - 追踪所有发布记录
- 🔒 **本地存储** - 数据安全，隐私保护

## 安装

### Windows
下载 `.exe` 安装包，双击运行安装。

### macOS
下载 `.dmg` 文件，拖拽到应用程序文件夹。

### Linux
下载 `.AppImage` 或 `.deb` 包安装。

## 使用说明

1. **添加平台账号** - 在「平台管理」中添加各平台的账号信息
2. **编写文章** - 在「写文章」页面创作内容
3. **一键发布** - 选择要发布的平台，点击「一键发布」

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build

# 打包发布
npm run dist
```
